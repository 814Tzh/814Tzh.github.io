原数组数据

let arr = [
    {id: 1, name: '部门1', pid: 0},
    {id: 2, name: '部门2', pid: 1},
    {id: 3, name: '部门3', pid: 1},
    {id: 4, name: '部门4', pid: 3},
    {id: 5, name: '部门5', pid: 4},
]

要求输出结果

[
    {
        "id": 1,
        "name": "部门1",
        "pid": 0,
        "children": [
            {
                "id": 2,
                "name": "部门2",
                "pid": 1,
                "children": []
            },
            {
                "id": 3,
                "name": "部门3",
                "pid": 1,
                "children": [
                    // 结果 ,,,
                ]
            }
        ]
    }
]

方法一

思路：根据对象调用的是内存地址，所以直接操作源数组，只需要把相互关联的子级放入父级，然后过滤出需要的结果即可。

//操作后原数组数据
[
    //部门1的children来源部门2、3
    {id:1,name:"部门1",pid: 0,children:Array(2)[{...},{...}]},
    {id:2,name:"部门2",pid: 1,children:Array(0)[]},
    //部门3的children来源部门4
    {id:3,name:"部门3",pid: 1,children:Array(1)[{...}]},
    //部门4的children来源部门5
    {id:4,name:"部门4",pid: 3,children:Array(1)[{...}]},
    {id:5,name:"部门4",pid: 4}
]

方法:

const arrayToTree =(arr,pid=0)=>{
    arr.forEach(itemA => {
        arr.forEach(itemB=>{
            if(itemB.pid===itemA.id){
                if(!itemA.children){
                    itemA.children = []
                }
                itemA.children.push(itemB)
            }
        })
    });
    return arr.filter((item)=>item.pid===pid)
}
console.log(arrayToTree(arr))

方法二（递归）

思路：

  • 首先创建一个空数组（resultArr）准备把结果放入其中
  • 根据id在源数组（arr）中查找其子项（pid===id），找到子项后为子项创建属性children:[ ]。
  • 将对象放入父项的children（可以把开始的空数组当成已经在一个children里）
  • 开始递归，然后已当前子项为新的节点，根据当前子项中的id在查找arr中对应子项的子项放入子项的children中

方法:

let resultArr=[]
const arrayToTree1 =(resultArr,arr,id=0)=>{
    arr.forEach((item)=>{
        if(item.pid===id){
            if(!item.children){
                item.children = []
            }
            resultArr.push(item)
            arrayToTree1(item.children,arr,item.id)
        }
    })
}
arrayToTree1(resultArr,arr)
