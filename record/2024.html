vue2作用域插槽 ​

vue
//data为子组件中的数据(data为自定义变量) 当前父组件中要使用该数据子组件得先暴露出来
<template slot-scope="data">
   {{data.obj.lastName}}
</template>

v-slot 指令自 Vue 2.6.0 起被引入，提供更好的支持 slot 和 slot-scope attribute 的 API 替代方案。v-slot 完整的由来参见这份 RFC。在接下来所有的 2.x 版本中 slot 和 slot-scope attribute 仍会被支持，但已经被官方废弃且不会出现在 Vue 3 中。也就是说，在vue2版本中，我们仍可以使用slot跟slot-scope，但是在vue3中就只能使用v-slot了。

uniapp编译微信小程序 v-for中插槽中内容只渲染一个元素 ​

问题复现：

子组件

vue
//u-swiper
<template>
	...
	<swiper-item class="u-swiper-item" v-for="(item, index) in list" :key="index">
        ...
        <slot />
        ...
	</swiper-item>
	...
</template>

父组件

vue
<u-swiper
  :list="posterList"
  img-mode="aspectFill"
  :height="posterHeight"
  bg-color="#fff0"
  :effect3d="true"
  border-radius="30"
  effect3d-previous-margin="100"
  :autoplay="false"
  @change="changesSwiper"
>
	//任意内容 微信小程序中只在第一个节点渲染
</u-swiper>

解决方案：

因为微信不支持v-for遍历普通slot，所以使用具名插槽 但是注意H5端生成很多次！！！

子组件

vue
//u-swiper
<template>
	...
	<swiper-item class="u-swiper-item" v-for="(item, index) in list" :key="index">
        ...
        <!-- 添加插槽 因为微信不支持v-for遍历slot 所以遍历插槽 H5会生成生成很多次！！ -->
		<!-- #ifdef MP-WEIXIN -->
		<slot name="{{index}}" />
		<!-- #endif -->
		<!-- #ifndef MP-WEIXIN -->
		<slot :index="index" />
		<!-- #endif -->
        ...
	</swiper-item>
	...
</template>

父组件（注意因为条件编译内容未被编译器识别，所以会出现报错）

vue
<u-swiper
  :list="posterList"
  img-mode="aspectFill"
  :height="posterHeight"
  bg-color="#fff0"
  :effect3d="true"
  border-radius="30"
  effect3d-previous-margin="100"
  :autoplay="false"
  @change="changesSwiper"
>
	   	<!-- 原组件未添加插槽 需手动在组件源码中添加 -->
        <!-- #ifdef MP-WEIXIN -->
        <view v-for="(item, index) in list" :key="index" slot="{{index}}">
          <view class="swiper-content">{{ list[index].title }}</view>
        </view>
        <!-- #endif -->
        <!-- #ifndef MP-WEIXIN -->
        <template v-slot="slotProp">
          <view class="swiper-content">{{ list[slotProp.index].title }}</view>
        </template>
        <!-- #endif -->
</u-swiper>

动态绑定多个值 模板语法 | Vue.js

如果你有像这样的一个包含多个 attribute 的 JavaScript 对象：

js
const objectOfAttrs = {
  id: 'container',
  class: 'wrapper',
  style: 'background-color:green'
}

通过不带参数的 v-bind，你可以将它们绑定到单个元素上：

vue
<div v-bind="objectOfAttrs"></div>

nextTick()

typescript
//类型
function nextTick(callback?: () => void): Promise<void>

也就是说可以使用promise相关语法

vue
import { nextTick } from 'vue'

async function increment() {
  count.value++
  await nextTick()
  // 现在 DOM 已经更新了
}
